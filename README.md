# Transformer UI

A cinematic ChatGPT concept landing page built with Vite, Three.js, and GSAP.

## Features

- Original transforming AI mech assembled from Three.js geometry
- Scroll-driven 3D choreography
- Pointer parallax and animated neural core
- Cinematic HUD, scanlines, particles, and terminal interface
- Responsive layout and reduced-motion support
- GitHub Pages deployment workflow

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The Vite base path is configured for the `transformer-ui` GitHub Pages project site.
